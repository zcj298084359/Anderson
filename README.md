# Anderson
H5活动页面（答题类游戏）

## 项目简介

安徒生童话主题的H5活动页面，答题不全对或超时跳转到各自页面可重玩，全对跳转到大转盘抽奖页面，抽奖后提交个人信息到后台，移动端项目，用电脑预览的话请切换到移动端。

## 技术栈

技术栈：HTML5、CSS3、jQuery
